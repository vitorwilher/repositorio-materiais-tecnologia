Fluxograma das trilhas

Mapa de todos os cursos, pré-requisitos e progressão

Este é o mapa completo do portfólio de Tecnologia: cada trilha como uma sequência de upsell (curso → próximo curso), os pré-requisitos, e as pontes entre trilhas (linhas tracejadas). Para o detalhe de cada jornada, abra a página da trilha correspondente.

Legenda de status: ativo no catálogo · novo (proposta a validar) · retorno (em atualização) · futuro (horizonte). Caixas tracejadas = ainda não publicado.

Mapa geral

Como ler o mapa

  • Setas cheias dentro de cada trilha = sequência de upsell (cada curso aponta para o próximo).
  • Setas tracejadas entre trilhas = pontes: pontos onde o aluno migra de uma trilha para outra (ex.: quem termina Python Avançado com IA na Programação pode entrar em SQL para Dados na trilha Dados).
  • Caixas azuis de destaque (Analista / Engenheiro de Dados) = profissões nomeadas ao fim da trilha.
  • O aluno-base Informática Fundamental com IA alimenta tanto a Trilha Dados quanto a de Produtividade.

Detalhe por trilha

Cada trilha tem seu próprio diagrama e tabela de cursos:

Trilha 1 — IA Aplicada · Trilha 2 — Dados · Trilha 3 — Infraestrutura · Trilha 4 — Cibersegurança · Trilha 5 — Programação · Trilha 6 — Produtividade

Atualizar este mapa: o fluxograma é escrito em Mermaid (texto) — para incluir um curso novo, edite o bloco ```{mermaid} acima (adicione o nó e a seta) e registre a mudança no Changelog. Veja o passo a passo em Como atualizar.
