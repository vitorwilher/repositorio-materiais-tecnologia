2. Requisitos de acesso

  • Idade mínima: 16 anos · Conhecimento específico: Windows e navegação na internet.
  • Documentos: identidade e CPF (cópia simples); comprovante ou autodeclaração do conhecimento específico.

3. Justificativa

Tecnologia foi um dos setores que mais contrataram no triênio 2020–2022, com forte demanda por desenvolvedores de aplicações web. Python é uma das linguagens mais promissoras (ciência de dados, web, apps, machine learning), usada por Google, Netflix, Uber, Spotify, Apple e Microsoft. O curso parte da lógica de programação — introduzida com Minecraft Education / MakeCode — até a capacidade de desenvolver aplicações web em Python.

4. Objetivo

Capacitar o aluno a desenvolver algoritmos com lógica de programação, estruturar aplicações front-end e desenvolver aplicações back-end para web com Python. Destinado a estudantes e interessados em ingressar na área de programação.

6. Orientações metodológicas

Metodologia de desenvolvimento de competências (Modelo Pedagógico Senac), com prática contextualizada. Por UC: UC1 — primeiro contato com lógica via Minecraft Education/MakeCode, depois na IDE; convite a egressos/profissionais. UC2 — wireframes a partir de sites reais, depois HTML + CSS responsivo; vivências de acessibilidade (simular uso por pessoas com deficiência); produto da UC conectado à UC3. UC3 — já integrar orientação a objetos; funcionalidades de aplicações web administrativas; uso de frameworks (Django) e integração com banco de dados; produto: aplicação web com BD, continuação da UC2.

7. Avaliação

Menção por indicador (A / PA / NA); por UC (Desenvolvida — D / Não desenvolvida — ND); curso (Aprovado — AP / Reprovado — RP). Aprovação exige D em todas as UCs e frequência mínima de 75%. Recuperação imediata à constatação das dificuldades.

8. Recursos, perfil e certificação

1 computador por aluno (i5 10ª geração+, 8GB, SSD 256GB, Windows 10 Pro) com Minecraft Education Edition, Visual Studio Code, Django e MySQL. Docente com experiência em desenvolvimento web — front-end (HTML, CSS, Bootstrap) e back-end (Python/Django), com ênfase em Minecraft Education. Certificado de Aperfeiçoamento em Programação em Python, validade nacional.

O PTD oficial é dividido por unidade curricular (três documentos):

  • PTD — UC1: Lógica de programação (.docx) — 48h / 12 aulas
  • PTD — UC2: Front-end para web (.docx) — 36h / 9 aulas
  • PTD — UC3: Back-end com Python (.docx) — 84h / 21 aulas

Situação de aprendizagem (comum às três UCs): o aluno entra na empresa fictícia JWC (desenvolvimento de aplicações web, Rio de Janeiro) num processo seletivo para Programador Full Stack, avançando por desafios — da lógica (UC1) ao portfólio front-end (UC2) e à aplicação back-end com banco de dados (UC3).

Recursos didáticos citados: Minecraft Education, IDE (Visual Studio Code e similares), Node.js, JavaScript, site-modelo desenvolvido pelo Senac (UC2). Avaliação atrelada a listas de atividades em aulas-marco de cada UC (ex.: UC1 — aulas 6, 10 e 12; UC2 — aulas 4 e 9).
