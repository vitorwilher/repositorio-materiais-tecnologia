2. Requisitos de acesso

  • Idade mínima: 16 anos · Conhecimento específico: Windows e navegação na internet.
  • Documentos: identidade, CPF e comprovante ou autodeclaração do conhecimento específico.

3. Justificativa

A tecnologia foi destaque de contratações no triênio 2020–2022, com forte demanda por desenvolvedores de aplicações web (crescimento do e-commerce). O PHP segue entre as linguagens mais usadas — estimado em cerca de 80% dos servidores — presente em Facebook, Wikipedia, Slack, Etsy e WordPress (43% da web). A lógica de programação é introduzida via Minecraft Education / MakeCode (Microsoft), abrindo oportunidades para o profissional atuar em empresa ou de forma autônoma.

4. Objetivo

Capacitar o aluno a desenvolver algoritmos com lógica de programação, estruturar aplicações front-end e desenvolver aplicações back-end para web com PHP. Destinado a quem quer ingressar no mercado de tecnologia e na programação PHP.

5. Organização curricular

Três unidades curriculares, 168h no total:

UC Unidade Curricular CH
UC1 Desenvolver algoritmos utilizando lógica de programação 48h
UC2 Desenvolver aplicações front-end para web 36h
UC3 Desenvolver aplicações back-end para web com PHP 84h

Elementos da competência por UC (conhecimentos / habilidades / atitudes):

UC Conhecimentos Habilidades Atitudes/Valores
UC1 Lógica de programação; algoritmos (tipos primitivos, variáveis, operadores, E/S); Minecraft Education / MakeCode; estruturas de controle (seleção, repetição, laços, contadores); vetores e matrizes (ordenação, pesquisa binária/sequencial, registros, arquivos indexados); testes (teste de mesa, debugadores); paradigmas (estruturada, modularização, funções, intro a OO); JavaScript (sintaxe e bibliotecas). Organizar conteúdo visual/textual; usar termos técnicos; pesquisar dados para apps web. Sigilo; zelo pela organização; respeito à propriedade intelectual; compromisso com usabilidade/acessibilidade.

6. Orientações metodológicas

Aprendizagem com autonomia e desenvolvimento de competências por situações contextualizadas. UC1: primeiro contato com lógica via Minecraft Education / MakeCode, depois IDE. UC2: produção de wireframes a partir de apps reais, HTML/CSS, frameworks responsivos e vivência de acessibilidade (simulação de restrições); produto integra com a UC3. UC3: programação já em OO, seleção de funcionalidades comuns a apps administrativos/comerciais, banco de dados orientado à prática e uso de frameworks; produto é uma aplicação web com banco de dados em continuidade à UC2.

7. Avaliação

Menções por indicador (A / PA / NA) e por UC (Desenvolvida — D / Não desenvolvida — ND). Aprovação no curso exige D em todas as UCs e frequência mínima de 75% (AP / RP). Recuperação imediata por situações-problema e estudos dirigidos.

8–11. Recursos, perfil e certificação

1 computador por aluno (i5 10ª geração+, Windows 10 Pro 64 bits, 8GB, SSD 256GB) com Minecraft Education, VS Code, Laravel, Composer e MySQL. Material Senac/aluno: não há. Docente com experiência em desenvolvimento web — front-end (HTML, CSS, JavaScript, Bootstrap) e back-end (PHP/Laravel) — e ênfase em Minecraft Education. Certificado de aperfeiçoamento em Programação em PHP, validade nacional.

Há três PTDs, um por unidade curricular:

  • Fonte oficial: PTD — UC1: Lógica de programação (.docx) — 48h / 12 aulas
  • Fonte oficial: PTD — UC2: Front-end (.docx) — 36h / 9 aulas
  • Fonte oficial: PTD — UC3: Back-end com PHP (.docx) — 84h / 21 aulas

Situação de aprendizagem (fio condutor das três UCs): a empresa fictícia de tecnologia JWC (desenvolvimento de apps web, sede no centro do Rio) abre uma vaga de Programador Full Stack com processo seletivo em etapas/desafios. O aluno assume o papel do candidato: aprovado, passa pelo período de experiência atendendo clientes da JWC.

UC Foco da situação de aprendizagem Recursos didáticos Avaliação
UC1 Desafios de seleção: lógica, estruturas de controle, vetores/matrizes e testes de algoritmos. Minecraft Education; IDE (VS Code e similares); Node.js; JavaScript. Listas de atividades das Aulas 6, 10 e 12.
UC2 Portfólio de aplicações web front-end no período de experiência (HTML, CSS, responsividade, acessibilidade). Site modelo desenvolvido pelo Senac; IDE (VS Code e similares). Atividades propostas nas Aulas 4 e 9.
UC3 Desafio final: desenvolver o sistema de e-commerce da JWC — ambiente, OO, requisições POST/GET, sessions/cookies, banco de dados, com Laravel. (ver PTD) Ao longo das 21 aulas.
