1. Identificação

  • Título: Analista de Dados — Big Data Science
  • Eixo: Informação e Comunicação · Segmento: Tecnologia da Informação
  • Tipo: Aperfeiçoamento · Carga horária: 120h
  • Código SGA: 13455 (cadastrado em 05.08.2024) · Código DN: 984.44

2. Requisitos e formas de acesso

  • Idade mínima: 18 anos.
  • Conhecimento específico: conhecimentos básicos em Excel.
  • Documentos para matrícula: documento de identidade e CPF.

3. Justificativa

Profissionais de análise de dados lideram as listas de carreiras mais demandadas. Em um mundo que gerou cerca de 40 trilhões de gigabytes de dados apenas em 2022, quem transforma dados em informações para a tomada de decisão tem alta demanda no mercado. Portais de vagas (LinkedIn, Indeed, Glassdoor) apontam o profissional de dados como um dos mais requisitados, com remuneração inicial atrativa — atraente tanto para quem migra de área quanto para quem quer aperfeiçoar o segmento de origem.

4. Objetivo

Dar subsídio ao participante no exercício do pensamento analítico e na interpretação de um conjunto de dados, possibilitando a manipulação de dados para gerar análises e insights que tragam valor à tomada de decisão ligada à área de negócios.

5. Organização curricular

O curso organiza-se em duas Unidades Curriculares (competências):

UC1 — Desenvolver algoritmos, versionamentos e linguagem de consulta estruturada (44h)

Elemento Conteúdo
Conhecimentos Lógica de programação; Python (conceito, ambiente, boas práticas); algoritmos (tipos, operadores, expressões lógicas e aritméticas); estruturas de controle (condicionais e loops); estruturas de dados (arrays, listas, bibliotecas); arquivos indexados e manipulação de tabelas; paradigmas de programação (estruturada, modularização, procedimentos, métodos e parâmetros); Git e GitHub (comandos, versionamento, repositório, fluxo colaborativo); biblioteca Pandas (manipulação de dados).
Habilidades Criar algoritmos para organização de dados; gerenciar ambiente de versionamentos.
Atitudes/Valores Colaboração no trabalho em equipe; responsabilidade e comprometimento com os acordos.
Indicadores Desenvolve algoritmos com lógica de programação; cria e manipula repositórios no GitHub conforme boas práticas de versionamento.

UC2 — Criar e manipular dados utilizando matemática estatística (76h)

Elemento Conteúdo
Conhecimentos Operações matemáticas e estatísticas com dados (média aritmética/ponderada, mediana, moda); Big Data (conceitos e tecnologias); banco de dados (modelo e relações, SQL/DQL — CRUD, desenvolvimento SQL); modelagem de negócio e computacional (requisitos, planejamento, execução, avaliação); Pandas (ETL, Series, pré-processamento, DataFrame); NumPy (arrays, medidas de tendência central e de dispersão, variância, desvio-padrão, correlação); Matplotlib e Seaborn (gráficos, histograma, boxplot); produtividade com IA (Pylance, Jupyter); visualização com Power BI.
Habilidades Planejar requisitos da análise estatística a partir da modelagem de negócio e computacional; aplicar matemática e estatística descritiva; interagir com IA generativa.
Atitudes/Valores Pensamento analítico; sigilo no tratamento de dados; zelo na manipulação de dados.
Indicadores Desenvolve análise estatística conforme requisitos; manipula base de dados com Python e bibliotecas; constrói dashboard com Power BI.

6. Orientações metodológicas

Percurso que privilegia a prática pedagógica contextualizada e a metodologia de desenvolvimento de competências (articulação de conhecimentos, habilidades e atitudes). Sugere-se o trabalho com bases de dados de domínio público (saúde, educação, segurança) para direcionar decisões. Alinha-se às Marcas Formativas Senac: domínio técnico-científico, visão crítica e atitude empreendedora, sustentável e colaborativa, com foco em resultados.

7. Aproveitamento de conhecimentos anteriores

Permitido conforme a legislação educacional vigente, mediante protocolo de avaliação de competências, desde que diretamente relacionados ao objetivo do curso.

8. Avaliação

Avaliação diagnóstica, formativa e somativa. Menções por indicador: Atendido (A) / Parcialmente atendido (PA) / Não atendido (NA). Menção por Unidade Curricular: Desenvolvida (D) / Não desenvolvida (ND). Para aprovação no curso o aluno precisa atingir D em todas as UCs e frequência mínima de 75% — Aprovado (AP) / Reprovado (RP). Recuperação imediata à constatação de dificuldades.

9. Instalações, equipamentos e recursos

1 computador por aluno (mínimo recomendado: Intel Core i5 10ª geração+, Windows 10+, 8GB RAM, internet 25Mb); laboratório com internet e projetor. Softwares: Microsoft Office 365, Visual Studio Code, bibliotecas Python. Recursos didáticos: não há.

10. Perfil docente

Docentes com experiência como Analista de Dados (desejável mínimo de 2 anos) e formação em TI ou áreas correlatas, preferencialmente com experiência em docência.

11–12. Bibliografia e certificação

Bibliografia: não há (no PC). Certificação: Analista de Dados — Big Data Science, validade nacional, para quem concluir com aprovação.

UC1 — Desenvolver algoritmos, versionamentos e linguagem de consulta estruturada (44h). O planejamento parte de situações de aprendizagem hands-on: estudo de caso do restaurante japonês (lógica e estruturação de problema), passo a passo do sanduíche (algoritmo), análise do vídeo “aprenda git e github” e pesquisa sobre funções e passagem de parâmetros. Recursos: laboratório com VS Code e GitHub Desktop. Avaliação registrada aula a aula em ficha individual.

UC2 — Criar e manipular dados utilizando matemática estatística (76h). Organizada em torno de uma situação de aprendizagem única ambientada na empresa fictícia “DataOrganized” (Curitiba), que lida com grandes volumes de dados de clientes em servidores na nuvem. As orientações gerais reforçam: testar atividades antecipadamente no laboratório, manter links/ferramentas de referência à mão, incentivar o uso de IA generativa como apoio à solução de problemas, e trazer enunciados próximos da realidade para desenvolver a compreensão de requisitos e a resolução de problemas.

Do PC. A seção de Bibliografia do Plano de Curso registra “Não há”. As ferramentas e tecnologias de referência do curso são: Python, Pandas, NumPy, Matplotlib/Seaborn, Git/GitHub, SQL, Power BI, Jupyter e Microsoft Office 365 / Visual Studio Code.
